# פרויקט פורטפוליו אישי

## סקירה כללית
מסמך זה מספק מדריך מקיף לאתר הפורטפוליו האישי שלי. הפורטפוליו מציג את הרקע המקצועי שלי, הכישורים והפרויקטים באמצעות ממשק אינטראקטיבי ומושך ויזואלית.

## אזור "קצת עליי"
אזור "קצת עליי" משמש כהקדמה למבקרים, מציג את הזהות המקצועית והרקע שלי. אזור זה מתוכנן כדי:

- להציג סיכום מקצועי תמציתי
- להדגיש מיומנויות וכישורים מרכזיים
- לגשת לקורות החיים שלי

האזור משתמש בעיצוב רספונסיבי כדי להבטיח צפייה אופטימלית בכל המכשירים, עם טיפוגרפיה ואלמנטי פריסה שנבחרו בקפידה המשקפים את הזהות המקצועית שלי.

## אזור הפרויקטים
אזור הפרויקטים מחולק לשני אזורים מובחנים, כל אחד מדגים יכולות טכניות וגישות פיתוח שונות.

### אזור 1: פרויקטים HTML/CSS
האזור הראשון מציג פרויקטים שפותחו באמצעות HTML ו-CSS בלבד. פרויקטים אלה מדגימים:

- טכניקות סגנון CSS מתקדמות
- עקרונות עיצוב רספונסיבי
- שליטה בפריסה באמצעות Flexbox .
- אנימציות ומעברים ב-CSS
- פתרונות תאימות בין דפדפנים

כל פרויקט באזור זה כולל תיאור, תצוגה מקדימה חזותית והסבר על האתגרים הטכניים שטופלו. היישום מדגיש HTML סמנטי ונקי בשילוב עם CSS מודולרי ומאורגן היטב.

### אזור 2: פרויקטים HTML/CSS/JavaScript
האזור השני מציג פרויקטים מורכבים יותר המשלבים פונקציונליות JavaScript לצד HTML ו-CSS. פרויקטים אלה מדגישים:

- רינדור תוכן דינמי
- אימות ועיבוד טפסים
- אינטגרציות API ופעולות אסינכרוניות

- הטופס בנוי מ-html ,css ,javaScript .
